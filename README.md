### рҹ§  Architecture

- рҹ–јпёҸ **Canvas compositing** вҖ” zoom, blur, backgrounds, webcam & ripples on one `<canvas>`
- рҹҺҷпёҸ **MediaRecorder API** вҖ” captures the composited canvas stream + audio tracks
- вҡЎ **Cached backdrop** вҖ” gradient + shadow re-rendered only when settings change
- рҹ•өпёҸ **Frame-differencing cursor** вҖ” lightweight fallback for tracking cursor in other windows
- рҹ§ӯ **SPA routing** вҖ” state-driven view switching, no React Router
- рҹҡ« **No backend** вҖ” everything runs client-side, nothing is ever uploaded
